PDF分段加载

使用PDF分段加载可以加快文档呈现速度，节省流量。

分段加载

pdfjsLib.getDocument({
url: ‘…’,
disableRange: false,
disableStream: true,
disableAutoFetch: true
…
})

使用说明

主要配置
disableRange = false
必须同时参数如下
disableStream = true （如果不配置，虽然分段加载生效，但浏览器会持续下载所有pdf文档）
disableAutoFetch = true
